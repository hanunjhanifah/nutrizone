GitHub Pages

Hosting gratis dari GitHub untuk proyek website statis. Cocok untuk pemula dan proyek personal.

  1. Buat akun GitHub
  2. Buat repository baru
  3. Upload file HTML, CSS, JS
  4. Aktifkan GitHub Pages
  5. Akses website via URL
Kunjungi GitHub Pages

Netlify

Platform hosting modern dengan deploy otomatis dari GitHub, GitLab, atau Bitbucket.

  1. Daftar akun Netlify
  2. Connect dengan GitHub
  3. Pilih repository
  4. Deploy otomatis
  5. Dapatkan URL custom
Kunjungi Netlify

Vercel

Platform hosting yang powerful dengan performa tinggi, cocok untuk website modern.

  1. Buat akun Vercel
  2. Import project dari GitHub
  3. Konfigurasi build settings
  4. Deploy dengan satu klik
  5. URL langsung aktif
Kunjungi Vercel

Hosting Berbayar

Hosting berbayar dengan fitur lengkap, domain gratis, dan performa optimal.

  1. Pilih paket hosting
  2. Daftar domain
  3. Upload file via cPanel/FTP
  4. Konfigurasi database (jika perlu)
  5. Website online dengan domain sendiri
Cek Hostinger

Firebase Hosting

Hosting dari Google dengan CDN global, SSL gratis, dan integrasi dengan layanan Firebase lainnya.

  1. Buat project di Firebase
  2. Install Firebase CLI
  3. Inisialisasi project
  4. Deploy ke Firebase
  5. Akses via URL Firebase
Pelajari Firebase

Cloudflare Pages

Platform JAMstack untuk frontend developer dengan deploy yang cepat dan global network.

  1. Daftar akun Cloudflare
  2. Connect dengan Git provider
  3. Pilih repository
  4. Setup build settings
  5. Deploy dan dapatkan URL

Cara Download & Simpan File Website

Langkah-langkah:

  1. Buka teks editor (VS Code, Notepad++, Sublime Text, dll)
  2. Salin kode HTML yang telah dibuat
  3. Buat file baru dengan nama index.html
  4. Tempel kode HTML ke dalam file
  5. Simpan file
  6. Buka file di browser untuk testing
Tips: Untuk testing lokal, cukup buka file index.html langsung di browser. Website akan berfungsi penuh meski tanpa server.

Struktur File yang Diperlukan:

project-folder/
├── index.html    (file utama)
└── assets/         (folder untuk gambar, dll - opsional)

FAQ - Pertanyaan Umum

1. Mana platform terbaik untuk pemula?

Netlify atau Vercel karena kemudahan penggunaan dan deploy otomatis dari GitHub.

2. Apakah website bisa diakses via smartphone?

Ya, semua platform di atas menghasilkan website yang responsive dan bisa diakses dari perangkat apapun.

3. Berapa biaya untuk hosting gratis?

Semua platform yang ditandai dengan tombol hijau benar-benar gratis untuk penggunaan dasar.

4. Bagaimana cara mendapatkan domain custom?

Anda bisa membeli domain dari penyedia seperti Namecheap, GoDaddy, atau langsung dari platform hosting.
